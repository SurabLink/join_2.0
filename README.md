# Kanban Project Management Tool

A simple Kanban-style task management application inspired by Trello.

## Description

This application allows users to manage tasks in a Kanban board with multiple columns such as:

- To Do
- In Progress
- Await Feedback
- Done

Users can organize their workflow efficiently by creating, updating, and tracking tasks across different stages.

The goal of this project was to practice building larger frontend applications with component-based architecture and user authentication.

## Features

- User registration and login (authentication)
- Create, edit, and delete tasks
- Drag & Drop functionality
- Task status management (To Do, In Progress, Await Feedback, Done)
- Assign contacts to tasks
- Create and manage contacts
- Summary dashboard with task overview and deadlines
- Responsive UI

## Technologies Used

- JavaScript
- HTML
- CSS
- Firebase
- Git
